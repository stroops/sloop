# ⚓ sloop

**AI-native terminal workspace manager**

> Terminal workspace manager for AI-native development

Create AI workspaces. Detach. Reattach later with full context preserved.  
Navigate your fleet of sessions without losing flow state or conversation history.

`sloop` is a thin CLI wrapper over a background daemon that manages persistent AI workspaces. 
Create, detach, attach, and navigate between AI sessions — each with preserved context, 
chat history, and embedded state — without losing your flow.

Think `tmux` meets AI context orchestration.
